MovieConsoleApp - Console Application for Managing Movies

This console application allows users to add movies to a JSON file, store them, view, and manage information about the movies, so they do not have to remember them but can store and manage their statuses.

How to Use

To use the application, run the command via the command line:

mvn exec:java

How to Build the Project

To build the project, use Maven:

mvn clean package

Dependencies

The application uses the following dependencies:

  • GSON 2.9.1 - for working with JSON files.

Java Version

The application is developed with Java version 21.

Developer Documentation

All code is documented using JavaDoc. Each method and package has appropriate comments.
